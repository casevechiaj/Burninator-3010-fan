# CAD .step file

- Use these files if you wish to modify something. 
- “Burninator_RAW.step” offers a clean base version of the Burninator, with no cosmetics, no spiral pattern, no extruder mounts, and no front LED logo, so you can use it as a foundation to build on.
- Keep in mind that the fan duct must blow air to the correct area: the tip of the hotend nozzle should sit about 1.5 mm beneath the lowest part of the cowl.
